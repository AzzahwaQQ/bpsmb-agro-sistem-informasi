# 🏛️ Sistem Informasi Pelayanan — BPSMB Agro Bandung

Aplikasi web manajemen pelayanan pengujian & kalibrasi 
untuk UPTD BPSMB Agro Bandung, dibangun selama 
Praktik Kerja Industri (PKL) 2025–2026.

## ✨ Fitur Utama
- Multi-role login (Admin & Front Office)
- Dashboard status permohonan real-time
- Sistem antrian digital
- Upload tanda tangan digital
- Generate & cetak dokumen PDF otomatis
- Manajemen user & role

## 🛠️ Teknologi
- PHP (Native)
- MySQL
- HTML / CSS / JavaScript
- Library FPDF (generate PDF)
